# House Price Prediction

This project predicts house prices using Machine Learning.

## Dataset
Custom House Prices Dataset

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib

## Model
Random Forest Regressor

## Features
- Overall Quality
- Living Area
- Garage Capacity
- Basement Area
- Year Built
- Bathrooms
- Bedrooms
- Lot Area

## Result
The model predicts house prices based on property characteristics.
